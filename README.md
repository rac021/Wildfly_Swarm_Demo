# Wildfly_Swarm_Demo


--------------------------------------------------------------

 - **Deployment**
 
``` 
     ❯  mvn -Dswarm.http.port=9090 wildfly-swarm:run

```     

 - **Test**
 
``` 
     ❯  curl http://localhost:9090/rest/hello
         
        $ Hello from WildFly Swarm Demo !
         
```     

 - ***Note :*** The default HTTP Port is **8080**
